Fixed Setups can be used to control certain aspects of the car setup for entrants. This guide explains how to create and set a Fixed Setup.

  • Load into the game and create the setup via the normal setup UI.
  • Save the setup in game, you can save it to a specific track or to general.
  • Find the saved setup file (should be in C:\Users\user-name\Documents\Assetto Corsa\setups\car-name\track-name, or you can open it from the car page in Content Manager). It will be called name-you-typed.ini.
  • If you want to unlock any values (for example tyres or fuel) remove their entries (name and value) from the .ini file.
  • Open the car page on the Manager (Cars & Setups Page, then pick the car).
  • Use the "Setup File" upload dialog to upload your fixed setup.
  • The setup should now be available in the "Fixed Setup" dropdown on each entrant in the entry list when creating an event.
